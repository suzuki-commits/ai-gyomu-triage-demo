# AI業務トリアージ ─ URLを入れるだけ（デモ）

企業サイトの URL を 1 つ入れると、その会社の主要業務を洗い出し
**🤖 自動化推奨 / 🤝 協働 / 👤 人間専任** の3分類 × 優先度に仕分けし、
実装ロードマップと株主総会レベルの提案 PPT を自動生成する AI ツールのデモです。

**株式会社コミクス（COMIX Inc.）** が開発。

## デモについて

このページは **デモ版** です。ボタンを押すと、実際に分析済みのサンプル
（株式会社コミクス自身の分析結果）を表示します。任意の URL でのライブ分析は、
ローカルで動作する本体ツール（`claude -p` 駆動）で実行します。

- 3分類テーブル／優先度マトリクス（売上インパクト×実装難易度）／3フェーズ実装ロードマップ／推奨アクション
- 生成済みの提案 PowerPoint（12ページ）もダウンロードできます

## 使い方

`▶ サンプル分析を見る` を押すか、URL を入れて `分析する` を押すと、
分析の様子（サイト調査 → 業務分類 → PPT 生成）を再生し、レポートを表示します。

## 構成

静的サイト（ビルド不要・依存なし）。

```
index.html    画面
style.css     スタイル（COMIX ブランドカラー）
app.js        デモ表示ロジック
sample.js     分析結果サンプル（コミクスの実データ）
decks/        ダウンロード用 PowerPoint
```
